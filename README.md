# BIN Checker
    Bank Identification Number Checker
    Data: https://binlist.net/

# Requirements
    pip install requests
    
# Clone Repo
    git clone https://github.com/Zeta0x/BIN-Checker.git
    cd BIN-Checker
    python lookup.py
